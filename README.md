# Blog application made with Django 

## Functionalities
    - View count
    - Search blog with title
    - Tag blogs 
    - WYSIWYG editor
    - Comments
    - Pagination

## Packages used:
    - Hitcount
    - Django Taggit
    - Django cirspy forms
    - Django tinymce
